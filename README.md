# Agung Sundoro Digital

Digital products marketplace showcasing courses and digital products by Agung Sundoro.

## Installation

1. Run `pnpm install` to install the dependencies.
2. Run `pnpm run dev` to start the development server.

## Production

1. Run `pnpm run build` to build the project.
2. Run `pnpm run start` to start the PM2 server on port 1012.

## PM2 Commands

- `pnpm run start` - Start the application
- `pnpm run stop` - Stop the application
- `pnpm run restart` - Restart the application
- `pnpm run delete` - Delete the PM2 process

## Development

Run `pnpm run dev` to start the development server on port 2012.

## Data

Products are loaded from `marketplace.json` in the public directory.
